Dataset Description


Train Data:

train.csv 파일 내의 데이터를 통해 모델을 훈련할 수 있습니다.
'label' 컬럼이 라벨이고, 이외 컬럼이 데이터입니다.

Test Data:

훈련한 머신러닝 모델을 기반으로 test.csv 내의 데이터를 분류하여 결과를 생성하시면 됩니다.
테스트 데이터의 결과는 제공되지 않으며, 리더보드에 업로드 시 채점되어 순위가 매겨집니다.
본 데이터는 어떠한 방식으로도 훈련에 활용될 수 없습니다.

submission-example.csv:

실제 리더보드에 제출하는 양식입니다.
해당 양식대로 제출해야 합니다. 해당 양식을 지키지 않으면 정확도가 다르게 채점될 수 있습니다.

Summary:

[train.csv]: 본인이 구현한 모델을 학습 및 평가할 수 있는 데이터셋입니다. 라벨이 포함되어 있습니다.
[test.csv]: prediction을 뽑아 제출용 csv 파일을 만들 데이터셋입니다. 라벨이 포함되어 있지 않습니다.
[sample_submissiom.csv]: Kaggle에 submission csv 파일을 제출하기 위한 템플릿입니다. 이와 같은 형태로 만들어 제출하시면 정확도가 산출되어 리더보드에 순위가 등록됩니다. (이와 다른 형태로 submission 파일을 작성할 경우 정확도가 제대로 계산되지 않으므로 주의해주세요.)
