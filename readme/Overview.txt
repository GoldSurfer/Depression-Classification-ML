Overview


Description:
Audio 및 Visual 특징 데이터를 활용하여 우울증 유무를 분류하는 프로젝트입니다.

주어진 dataset은 우울증을 앓고 있는 사람의 영상에서, 해당 영상의 발화자의 얼굴에서 뽑은 여러가지 특징과(i.e. visual features) 발화자의 목소리에서 뽑은 여러가지 특징(i.e. audio features)을 담고 있습니다.
'label' 컬럼에는 우울증에 걸린 경우(= 1)와 그렇지 않은 경우(= 0)에 대하여 라벨링 해놓은 정보가 담겨 있습니다.

본 과제는 다양한 feature들이 주어졌을 때(e.g. emotion_ angry_ mean, emotion_ disgust_ mean, etc.), 해당 데이터가 어떤 라벨인지를 예측하는 Classification(분류) 문제를 푸는 것이 본 Competition의 목적입니다.
우울증 유무를 분류하는 머신러닝 모델을 설계하시는 프로젝트로, 사이킷런 등의 라이브러리에서 구현된 모델을 사용하여(또는 직접 스크래치로 구현 가능, 이미 구현된 모델 구조 튜닝 가능) 학습 및 평가하셔야 합니다.
개인이 구현한 모델로 주어진 데이터셋에 대해 학습 및 예측한 결과의 정확도로 경쟁을 진행하게 됩니다.

Evaluation:
순위 경쟁을 통해 점수를 차등 부여(상대평가)합니다.
테스트 데이터에 대해서 80%만을 활용하여 스코어보드에 Public Rank로 보여줍니다.
20%는 스코어보드에는 보여지지 않지만, 종료 후 확인이 가능한 Private Rank 측정에 활용됩니다.
Public Rank과 Private Rank를 함께 고려하여 최종 점수가 산출됩니다.
Public Rank에 활용되는 데이터에 오버피팅 되지 않도록 강건한 모델을 제작해야 합니다.

Submission:
1일 최대 5회, Private 등록은 1회까지 가능합니다. (테스트 데이터에 오버피팅 하는 것을 막기 위해)

Rules:
- 컴페티션 웹페이지에서 데이터를 다운로드하고, Train 데이터를 활용하여 머신러닝 모델을 학습한 후 Test 데이터를 분류
- 사이킷런 사용 가능하며 메소드 제한 없음. 단, 딥러닝 모델 사용 금지
- 순위에 따라 차등적으로 점수 부여
- 성능을 올리기 위해 어떠한 자료와 인공지능을 활용해도 좋습니다. 단, 어떤 아이디어를 사용했는지, 왜 해당 소스를 활용했는지 설명할 수 있어야 합니다.
- 머신러닝 방법을 사용해야하며, 코드와 해당 결과 CSV를 추출하기 위해 사용한 코드 제출을 요청 받을 수도 있습니다.
