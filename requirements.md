# Requirements

## Python Version
- Python >= 3.8

## 필수 패키지
다음은 프로젝트 수행에 필요한 주요 파이썬 패키지입니다:

```
pandas>=1.3.0  
numpy>=1.21.0  
scikit-learn>=1.0  
matplotlib>=3.4  
seaborn>=0.11  
```

## 설치 방법

아래 명령어를 통해 필요한 패키지를 한 번에 설치할 수 있습니다:

```bash
pip install -r requirements.txt
```

## requirements.txt 예시

```

```
